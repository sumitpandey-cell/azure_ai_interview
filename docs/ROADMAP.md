# Feature Roadmap & Versioning
## AI Interviewer Platform

### 1. Versioning Strategy
We use Semantic Versioning (SemVer) `MAJOR.MINOR.PATCH`.
- **v1.0.0 (MVP)**: Current Release. Core value proposition functioning.
- **v2.0.0 (Planned)**: Major platform expansion with new user flows.

### 2. Feature Comparison Matrix

| Feature Category | MVP (Current v1.0) | V2 Enhancements (Planned) |
| :--- | :--- | :--- |
| **Interview Types** | Voice-only AI Interviews | Video Analysis (Body Language, Eye Contact) |
| **Templates** | 20+ Pre-defined Role Templates | Custom Template Builder for Enterprise |
| **Feedback** | Text-based Performance Reports | Multi-modal Feedback (Audio snippets, Video clips) |
| **Auth** | Email/Pass, Google OAuth | LinkedIn Integration, SSO for Teams |
| **Analytics** | Individual User Progress | Team/Organization Dashboards |
| **Resume** | Manual Skill Entry | Auto-extraction from PDF/Docx Upload |
| **Collaboration** | Solo Practice | Multiplayer / Mock Interviews with Peers |
| **Mobile** | Responsive Web App | Native iOS/Android App |

- [ ] **Q3 Month 2**: Implement "Resume Parsing" to auto-suggest interview templates.
